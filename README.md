# coursechecker
Checks existing courses offered each quarter and updates attached excel. 
1) Upload a .xlsx file with the department in column A and course number in column B
2) Program will automatically prompt you to enter your desired quarter and year to check: 'Spring 20XX', 'Autumn 20XX', or 'Winter 20XX'
3) Will check courses listed in the uploaded excel file against the scheduling website

Program returns your original file and whether the course is offered that quarter indicated by a Y for 'Yes'.
